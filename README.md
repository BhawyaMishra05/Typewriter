# Typewriter

A nostalgic digital typewriter that recreates the feel of writing on a vintage typewriter with modern web technologies.

## Overview
The project simulates a realistic typing experience through animated keys, customizable themes, and paper-style editing.
It focuses on recreating the distraction-free writing experience while adding useful modern features.

## Features
- Realistic typewriter animation
- Multiple ink colors
- Theme switching
- Paper-style editor
- Keyboard sound simulation
- Download text
- Responsive interface

## Tech Stack
- HTML
- CSS
- JavaScript

## Future Scope
- Cloud document storage
- Markdown support
- Export as PDF
- Writing statistics
- Collaborative editing
